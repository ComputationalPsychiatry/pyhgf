pyhgf.model.DeepNetwork#

class pyhgf.model.DeepNetwork(kind='volatile-state', coupling_fn=(<PjitFunction of <function tanh>>, ))[source]#

A deep network with fully connected layers for predictive coding.

This class extends Network with automatic layer tracking and convenience methods for building, visualizing, and training deep hierarchical networks.

Unlike the base Network class, DeepNetwork automatically tracks layer structure as you add layers, providing simplified interfaces for plotting and fitting.

Examples

>>> # Build a deep network with method chaining
>>> net = (
...     DeepNetwork()
...     .add_nodes(kind="continuous-state", n_nodes=10)
...     .add_layer(size=8, precision=1.0)
...     .add_layer(size=6, precision=1.0)
...     .add_layer(size=4, precision=1.0)
... )
>>>
>>> # Or use add_layer_stack for multiple layers at once
>>> net = (
...     DeepNetwork()
...     .add_nodes(kind="continuous-state", n_nodes=10)
...     .add_layer_stack(layer_sizes=[8, 6, 4], precision=1.0)
... )
>>>
>>> # Plot without specifying layer indices
>>> net.plot_layers()
>>>
>>> # Fit with automatic input/output detection
>>> net.fit(x_train, y_train)
